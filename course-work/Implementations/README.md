# SafeBG — Система за подаване на сигнали

**Факултетен номер:** 2301261070  
**Име на проекта:** SafeBG  

##  Описание
SafeBG е уеб приложение, което позволява на граждани да подават сигнали за проблеми в градската среда.  
Системата поддържа три нива на достъп — Guest, User и Admin — и предлага функционалности като създаване на сигнали, коментиране, гласуване, филтриране и административно управление.

##  Технологии
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server LocalDB
- Identity (потребители и роли)
- Bootstrap 5
- Leaflet.js (карта за избор на локация)

##  База от данни
Проектът използва **минимум 3 свързани таблици**:
- Reports
- Comments
- Votes
- Cities
- Categories
- AspNetUsers (Identity)

Таблиците имат различни типове данни: int, string, DateTime, bool.

##  Роли в системата
- **Guest:** вижда само списък със сигнали  
- **User:** може да създава сигнали, да гласува и коментира  
- **Admin:** пълен достъп — редакция, изтриване, промяна на статуси  

##  Основни функционалности
- ✔ Създаване на сигнал с карта (Leaflet)
- ✔ Филтри по град, категория, статус и ключова дума
- ✔ Подреждане по дата, град, категория
- ✔ Коментари към сигнал
- ✔ Like / Dislike на коментари
- ✔ Гласуване на сигнал (1 потребител → 1 глас)
- ✔ Администраторски панел за редакция/изтриване
